## 📌 What It Does
Paste any content (blog post, article, newsletter, idea) and instantly get:

| Platform | Format |
|----------|--------|
| 🐦 Twitter/X | Thread (numbered tweets, max 280 chars each) |
| 📸 Instagram | Caption with emojis and hashtags |
| 💼 LinkedIn | Professional storytelling post |
| 🎬 Reels/TikTok | Script with [VISUAL] and [TEXT] cues |
| ✈️ Telegram | Conversational post with markdown |

## ⚡ Key Features
- **Single API call** → 5 outputs simultaneously
- **Real-time generation** via Groq (world's fastest LLM inference)
- **Copy to clipboard** per platform with confirmation
- **Regenerate** any output instantly
- **Character count** with color-coded limit indicator per platform
- **Fully responsive** — works on mobile
- **Zero backend** — pure frontend, serverless

## 🛠️ Tech Stack
- **Frontend:** React + Vite + Tailwind CSS
- **AI Engine:** Groq API (LLaMA 3.3 70B)
- **Deployment:** Bolt.host
- **Version Control:** GitHub

## 🔧 Setup & Run Locally

```bash
# Clone the repo
git clone https://github.com/Tama260/AI-Repurposing-Engine.git

# Install dependencies
cd AI-Repurposing-Engine
npm install

# Set environment variable
echo "VITE_GROQ_API_KEY=your_groq_api_key_here" > .env

# Run development server
npm run dev
```

## 🔑 Environment Variables
| Variable | Description |
|----------|-------------|
| `VITE_GROQ_API_KEY` | Your Groq API key from [console.groq.com](https://console.groq.com) |
